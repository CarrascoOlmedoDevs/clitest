Vamos a empezar a construir el simulador de fútbol. Tu tarea es crear los ficheros con las clases y configuraciones fundamentales.

Por favor, realiza los siguientes pasos:

1.  **Crea el fichero `config.py`** en el directorio `clitest` con el siguiente contenido:
    ```python
    # clitest/config.py
    POSICIONES = {
        "PO": "Portero",
        "DEF": "Defensa",
        "MED": "Mediocentro",
        "DEL": "Delantero"
    }

    DURACION_PARTIDO = 90
    ```

2.  **Crea el fichero `player.py`** en el directorio `clitest` con el siguiente contenido:
    ```python
    # clitest/player.py
    class Player:
        def __init__(self, nombre, posicion, stats):
            self.nombre = nombre
            self.posicion = posicion # Ej: "PO", "DEF", "MED", "DEL"
            self.stats = {
                "tiro": stats.get("tiro", 50),
                "pase": stats.get("pase", 50),
                "velocidad": stats.get("velocidad", 50),
                "defensa": stats.get("defensa", 50),
                "parada": stats.get("parada", 50) # Solo para porteros
            }

        def __str__(self):
            return f"{self.nombre} ({self.posicion})"
    ```

3.  **Crea el fichero `team.py`** en el directorio `clitest` con el siguiente contenido:
    ```python
    # clitest/team.py
    from player import Player

    class Team:
        def __init__(self, nombre, jugadores):
            self.nombre = nombre
            if len(jugadores) != 11:
                raise ValueError("Un equipo debe tener exactamente 11 jugadores.")
            self.jugadores = jugadores

        def __str__(self):
            return self.nombre
    ```

4.  **Sube tus cambios al repositorio.** Ejecuta los siguientes comandos de Git en orden:
    ```bash
    git add config.py player.py team.py
    git commit -m "Agente 2: Añade clases Player, Team y fichero de configuración"
    git push origin main
    ```
